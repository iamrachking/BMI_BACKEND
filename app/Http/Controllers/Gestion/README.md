# Module Gestion (Équipements Industriels)

Contrôleurs Blade pour la gestion des équipements industriels, maintenances et pannes.

## Contrôleurs à créer :

- `EquipmentController.php` - CRUD équipements
- `MaintenanceController.php` - Gestion des maintenances (planification, suivi)
- `FailureController.php` - Historique des pannes
- `DashboardController.php` - Tableau de bord (machines actives, en panne, en maintenance)

## Routes préfixées : `/gestion`
